Welcome to GoPPydae 🐠

GoPPydae is the family of programs built on Go, Protocol Buffers, and Python.

We build performant, scalable, and maintainable tools that play to each language's strengths, from high-speed backend services to heavy data processing.

Stack

Core Technologies

go

Go

Powering high-concurrency backend services, CLI tools, and systems that demand raw performance.

protobuf

Protocol Buffers

Defining strictly typed, language-agnostic communication contracts shared across every service in the ecosystem.

python

Python

Handling data processing, scripting, automation, and rapid prototyping across the ecosystem.

Communication is anchored by a centralized .proto file defining shared data structures and service contracts, from which language-specific message types, gRPC interfaces, and client stubs are generated for both languages. Python uses the standard protobuf library from PyPI; Go uses Google's official protobuf modules, giving the ecosystem a consistent serialization foundation.
